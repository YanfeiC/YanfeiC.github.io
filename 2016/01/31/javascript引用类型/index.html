Object类型

创建Object实例的两种方式

  1. 使用new操作符
  1. 对象字面量

访问对象属性的两种方法

  1. person.name
  2. person[“name”]
    []括号内的参数可以是变量
    var x = “name”;
    person[x];

Array类型

创建数组

  1. var x = new Array();
  2. 省略new，var x = Array();
  3. 使用数组字面量，var x = [1,2,3,"hello"];

length属性

数组length属性可以获取数组的长度，主动设置length可以增加数组长度或移除多余项。

检测是否是数组的函数

Array.isArray(value);

转换数组为字符串

  • toString()
  • toLocalString()
  • valueOf()
  • join() 上面都是使用逗号来分隔字符，join可以自定义分隔符。x.join("|")

栈方法

  • push() 接受任意数量参数，添加到数组末尾，返回修改后的数组长度。
  • pop() 移除数组末尾最后一项，返回移除的值。

队列方法

  • shift() 移除数组的第一项并返回该项。
  • push() 同上
  • unshift() 在数组前添加任意项。

排序方法

  • reverse() 反转数组顺序
  • sort() 按从小到大排序
    sort()函数可以接受一个比较函数，它接受两个参数，如果参数的顺序符合排序的顺序，则返回负数。相等返回0，否则返回正数。

操作方法

  • concat()
    基于当前数组建立新数组，参数可以是数值或者是数组。

  • slice()
    返回切分后的数组

  • splice()

    • 删除 数组中指定位置。需要2个参数，要删除的第一项位置和要删除的项数。splice(2,0);
    • 插入 在指定位置插入数据。需要3个参数。起始位置，0(要删除的项数)，要插入的项。splice(2,0,"red","white");
    • 替换 在指定位置插入任意数量的项，且同时删除任意数量的项。需要3个参数。起始位置，要删除的项数，要插入的项。splice(2,1,"red","green");
    • splice()返回删除的项。

位置方法

该方法可返回某个指定的字符串值在字符串中首次出现的位置。

  • indexOf()
  • lastIndexOf()
    接受两个参数
参数 描述
1 必需。规定需检索的字符串值.
2 可选的整数参数。规定在字符串中开始检索的位置。它的合法取值是 0 到 stringObject.length - 1。如省略该参数，则将从字符串的首字符开始检索。

迭代方法

  • every()
  • some()
  • filter()
  • forEach()
  • map()

归并方法

  • reduce()
  • reduceRight()

Date类型

Date 实例化

var now = new Date();获取当前时间
可以指定一个日期创建对象，需要传入表示改时间的毫秒数，为了简化计算过程,有两个方法。

  • Date.parse()
    接受一个表示日期的字符串，使用方法
    var someDate = new Date(Date.parse("May 25, 2004"));
  • Date.UTC()

Date.now()
返回调用这个方法时的毫秒数，这个函数可以轻松检测一些操作需要的时间

函数类型

函数定义

  • 函数声明

  • 函数表达式

  • js解析器会先读取函数声明，并使其在执行任何代码之前可用；而函数表达式，则等到执行它所在的代码行才会真正解析。所以如果函数调用在函数声明前，会正常运行。而通过函数表达式定义函数的话，会抛出错误。

    arguments是一个类数组对象，包含着所有传入函数的参数。这个对象有一个叫callee的属性，该属性是一个指针，指向拥有arguments对象的函数。

函数属性和方法

  • 属性

    • length
      表示函数希望接受的命名参数的个数
    • prototype
      prototype是引用类型保存它们所有实例方法的地方。

    • apply()和call()
      在特定的作用域调用函数。作用如下：

      1. 传递参数

      2. 扩充函数赖以运行的作用域。

      3. bind()
        bind()方法会创建一个函数实例，其this值会绑定到传给bind()函数的值。

基本包装类型

为了操作基本类型值，ECMAScript提供了3个特殊的引用类型，Boolean、Number和String.每当读取一个基本类型值后，后台都会创建一个基本包装类型的对象，从而让我们能够调用一些方法来操作这些数据。

String类型

字符方法

  • charAt()
    接收位置参数，返回指定位置的字符。

  • charCodeAt
    接收位置参数，返回指定位置的字符编码。

字符串操作方法
  • concat()
    连接字符串，多数情况可以使用＋号操作符。
  • slice() substr() substring()
字符串位置方法
  • indexOf() lastIndex()
    搜索字符串，返回字符串位置

找到所有的匹配的字符串

trim()方法

删除字符串前置和后置所有空格。

大小写转换
字符串匹配
  • match()
  • search()
  • replace()

单体内置对象

Global对象

URI编码
  • encodeURI()
    主要用于整个URI编码

  • encodeURIComponent()
    主要用于对URI的某一段进行编码。

  • decodeURI()
