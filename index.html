TestNG测试不更新数据库

发表于 2017-02-04   |   分类于 Java   |  

最近在看精通Spring4.x一书,在2.4.3节单元测试这一节,书中演示了使用TestNG对UserServie的单元测试.
第三个测试方法testAddLoginLog(),测试了添加t_login_log数据的过程.书中说明这个方法执行后可以在此表中添加一条登陆记录,然而并没有!!!
查了很长时间,代码应该没问题,使用网页实测会有登陆记录.原来是TestNG在进行数据插入时会进行数据回滚,从而不影响数据库的内容.

如果希望关闭回滚操作可以使用@Rollback声明

当然一般情况下并不需要关闭它,毕竟我们并不希望测试方法老是往数据库添加无用数据.
